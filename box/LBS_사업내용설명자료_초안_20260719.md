## 2. 서비스 개요
- **서비스명**: 도시공존 (dosigongzon.com, 웹앱 및 Google Play 스토어 등록 앱)
- **서비스 성격**: 길고양이 돌봄 시민 참여 플랫폼 — 지역 주민이 동네 길고양이를 등록하고 돌봄(급식 등) 기록을 남기며, 주변 동물병원·동물약국 정보를 지도로 제공
- **위치정보사업자(측위 제공자)**: 이용자 단말기의 GPS (브라우저 Geolocation API 경유), 지도 표시는 카카오맵 API 사용

## 3. 위치정보의 이용 목적 및 방식 (총 3가지)

| # | 기능 | 수집 항목 | 이용 방식 | 저장 여부 |
|---|---|---|---|---|
| 1 | 홈 화면 날씨·돌봄 팁 | 단말기 현재 좌표(위도·경도) | 좌표를 서버에 전송해 해당 지역 날씨 조회 후 즉시 응답 | **저장하지 않음** (조회 즉시 폐기) |
| 2 | 지도 "내 위치로 이동" | 단말기 현재 좌표 | 클라이언트(브라우저) 내에서 지도 중심 이동에만 사용 | **서버 미전송·미저장** |
| 3 | 고양이 등록 시 동네 자동 감지 | 단말기 현재 좌표 | 행정동 단위로 변환해 등록 지역 제안 | 원좌표 미저장, **동 단위로 퍼징(fuzzing)된 좌표만 저장** |

- 개인위치정보를 **식별 가능한 형태로 축적·보관하지 않음** (위 표의 저장 정책).
- 위치정보 수집 전 브라우저/OS 표준 권한 요청으로 **이용자 동의**를 받으며, 거부 시에도 서비스 이용 가능(날씨는 IP 기반 대체).

## 4. 개인위치정보 보호 조치
- 이용약관 제9조의5에 위치정보 수집 항목·목적·보유(미저장)·제3자 제공(없음)·이용자 권리·관리책임자를 고지
- 전 구간 HTTPS 암호화 전송, 데이터베이스 접근은 행 수준 보안(RLS) 적용
- 개인위치정보의 제3자 제공 없음, 광고 목적 이용 없음
- 위치정보 관리책임자: 대표자 (연락처 상동)

## 5. 소상공인 해당 여부
- 「소상공인 보호 및 지원에 관한 법률」 제2조에 따른 소상공인 (상시근로자 0명, 1인 사업자)
- 증빙: 소상공인확인서 첨부 (중소기업현황정보시스템 발급)
