### TRD에서 함께 제거되는 항목

**Domain Layer:**

- `LocationSecurityService` (위치 보안 서비스)
- `Location VO` (좌표 Value Object)
- `AuthLevel VO` (Lv.1/2/3 인증 등급)
- `CareLog Entity`
- `Cat Entity`

**Application Layer:**

- `ResolveLocationUseCase`
- `CreateEmergencyPostUseCase`
- `LockdownRegionUseCase`
- `AiVetConsultUseCase`
- `CreateCareLogUseCase`

**Infrastructure Layer:**

- `ai-guardrail.adapter.ts`
- `gemini-ai.adapter.ts`
- `supabase-realtime.adapter.ts`
- Kakao Maps Adapter

**DB 스키마:**

- `cats` 테이블
- `care_logs` 테이블
- `ai_consult_logs` 테이블
- `posts` 테이블의 좌표 컬럼 (`latitude`, `longitude`, `blurred_lat`, `blurred_lng`)

**외부 의존성 제거:**

- Gemini API
- Cloudflare Stream
- Kakao Maps SDK

## 2. 남는 기능 — 소규모 커뮤니티 MVP

| 기능 | 상세 |
|------|------|
| **소셜 로그인** | Kakao/Google (Supabase Auth). Lv 구분 없이 로그인 = 전체 권한 |
| **게시판** | 카테고리별 글 목록 + 상세 + 작성 + 수정 + 삭제 |
| **댓글** | 게시글에 댓글 CRUD |
| **이미지 업로드** | 게시글 첨부 이미지 (Supabase Storage) |
| **신고 + 자동 블라인드** | 3회 누적 시 블라인드 |
| **관리자 공지 고정** | 관리자가 특정 글 상단 고정 |
| **도움센터** | SSG 정적 가이드 (냥줍, 응급, 포획, 법률 등) |

### 커뮤니티 카테고리 (9개, 축소 검토 가능)

1. 긴급 제보
2. TNR 정보 공유
3. 급식소/쉼터 관리
4. 임시보호/입양
5. 동물병원 후기/추천
6. 법률/행정 Q&A
7. 자유게시판
8. 성공 스토리
9. 지역별 소모임

> **미결 사항:** 지역 소모임 처리 방식 (제거 vs 태그 수준 단순화) 미정.
