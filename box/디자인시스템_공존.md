# 공존 디자인 시스템 (2026-07-15)

토스풍의 클린·미니멀 감성을 따르되, **트레이드드레스(고유 외형) 침해를 피하기 위해
'서명 요소'를 의도적으로 다르게** 만든 디자인 언어. 핵심 차별점은 **원형 → 라운드 사각**.

## 왜 이렇게?
- 토스의 보호 대상은 특정 색·아이콘셋·일러스트(Toss Face)·전체적 외형 조합.
- 색상 hex(#3182F6 등)는 상표 대상이 아니고, 우린 고양이 돌봄앱 ↔ 토스는 금융앱이라
  소비자 혼동 가능성이 낮음. 그래도 안전하게 형태 언어를 분명히 다르게 간다.

## 서명 요소 (토스와 일부러 다르게)
| 요소 | 토스 경향 | 공존 |
|---|---|---|
| 토글 노브 | 원형 | **라운드 사각** (radius-square-sm) |
| 토글 트랙 | 완전 pill | **라운드 사각** (radius-square-lg) |
| 칩·태그 | 풀 라운드 pill | **라운드 사각** (radius-square, 8px) |
| 선택/활성 배지 | 원형 도트 | 사각 or 발자국(🐾) 모티프 |
| 아이콘 | 자체 아이콘셋 | lucide-react + 브랜드 순간엔 🐾 |
| 브랜드 마크 | Toss Face | 고양이 발자국 |

일반 카드/시트는 기존 곡선(radius-card 20px 등) 유지 — 부드러움은 살리고,
"토글·칩" 같은 컨트롤에서만 사각 시그니처를 준다.

## 토큰 (globals.css)
- 색: `--color-primary`(공존 블루 #3182F6), text/surface/border 뉴트럴 스케일.
- radius(곡선): input 14 / card-sm 16 / card 20 / modal 24 / sheet 28 / full.
- **radius(사각 시그니처)**: `--radius-square-sm` 6 / `--radius-square` 8 / `--radius-square-lg` 10.
- shadow: card-sm / card / raised / primary / modal.

## 컴포넌트 (app/components/ui/)
- **SquareToggle** — 독립 토글(설정 등). 이메일/마케팅 설정 토글 적용 완료.
  라벨 행 전체가 버튼인 경우(비밀글 등)는 중첩 버튼 방지 위해 노브만 사각 인라인 처리.
- **Button** (2026-07-15) — primary/weak/neutral/text/danger × lg/md/sm. pill 금지, .press 내장.
- **Card** (2026-07-15) — 링크/버튼/정적 3용법. 탭 가능하면 .press 자동.
- **ListRow** (2026-07-15) — [아이콘박스] 제목/부제 ─ 우측값 > 문법. 아이콘 박스 라운드 사각.
- (예정) SquareChip, SquareBadge — 순차 도입.

## 타이포·모션 (2026-07-15 추가)
- 타이포 스케일 6단계: display 24 / title 20 / section 17 / body 15 / label 13 / caption 11.
  섹션 헤더는 색 막대 장식 없이 17px extrabold만으로 위계 (홈 적용 완료).
- 모션: `--motion-fast` 0.15s / `--motion-base` 0.2s / `--motion-spring` 0.3s.
- 프레스 피드백: `.press`(scale 0.98 + opacity) / `.press-strong`(0.96) — 탭 가능한 모든 표면.
