## 1. 지도 UX 개선 (호갱노노 참고)

### 헤더 리디자인
- 큰 갈색 배너(4줄) → **슬림 한 줄** 헤더
- `[서구 2/6 | 오늘 방문자 105명] [고양이] [병원] [약국]`
- 동 단위 안내 배너 삭제 (게스트 배너에서 이미 안내)

### 필터 칩
- 🐱 고양이 / 🏥 병원 / 💊 약국 텍스트 토글
- 활성: 색상 배경 + 흰색 글씨 / 비활성: 흰색 + 회색

### 줌 레벨별 병원 표시
| 줌 | 표시 |
|----|------|
| 12+ | 전부 숨김 |
| 9~11 | 약국 + 수동 등록만 |
| 7~8 | + 추천(pinned) 병원 |
| 6 이하 | 전부 표시 |

### 기타
- 내 위치 GPS 버튼 (우측)
- 고양이 마커 z-index 10 (최상위)
- 안내 말풍선: 등록/채팅 유도 문구
- 하단 버튼 위치 조정 (BottomNav 겹침 해결)

## 3. 비회원 지도 모드

### 흐름
- 첫 방문 → 온보딩 → `/map` (기존: `/login`)
- 재방문 비로그인 → 홈 → `/map` 자동 이동

### 비회원 제한
- 고양이 마커 보기: ✅ (70m 오차)
- 고양이 클릭: 🔒 로그인 유도 confirm
- 동 클러스터 클릭: 🔒
- + 등록: 🔒
- 동네 채팅: 🔒
- 상단 배너: "둘러보기 모드" + 로그인 버튼

### 비회원 방문자 카운트
- IP 기반 하루 1회 (인메모리 중복 방지)
- 지도 페이지 진입 시 자동 카운트
