  - 서울 접속 → 서울 날씨, 부산 접속 → 부산 날씨

## 11. AI 집사 안정성 대폭 강화

### 발생한 문제들
1. **429 Too Many Requests** — 무료 티어 분당 쿼터 초과
2. **503 Service Unavailable** — Gemini 서버 과부하
3. 기존 코드는 429/503 에러 시 폴백 없이 에러 표시

### 해결
- **모델 6개로 확대**: flash-lite → flash → 2.5-flash → 2.5-flash-lite → flash-lite-latest → flash-latest
- **429/503도 다음 모델로 자동 폴백** (기존: 404만 폴백)
- **오프라인 키워드 매칭 폴백**: 6개 모델 전부 실패 시 로컬에서 즉시 답변
  - 밥/사료, 숨숨집/온도, 부상/병원, TNR, 새끼고양이, 학대, 물 7개 주제
  - 키워드 미매칭 시 보호지침 안내 + 동물보호콜센터 번호
- **서버 로그** 추가: 어떤 모델이 시도/성공/실패했는지 터미널 확인 가능

### 결과
- AI 집사가 **절대 에러 메시지를 보여주지 않음**
- Gemini API 가용 시 → AI 답변 / 불가 시 → 오프라인 답변

## 14. 교훈 & 메모

- **HTTP에서 GPS 차단**: 모바일 Chrome은 HTTPS가 아니면 `navigator.geolocation` 차단 → IP 기반 위치 추정으로 해결
- **Gemini 무료 티어 한계**: 분당/일일 쿼터가 모델별로 따로 적용됨 → 여러 모델 폴백이 효과적
- **에러는 삼키지 말 것**: 사용자에게 에러를 보여주되, 최종 폴백은 반드시 마련할 것 (오프라인 답변)
- **placehold.co**: 개발 중 임시 이미지로 유용 (색상/텍스트 지정 가능)
