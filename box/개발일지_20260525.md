## 핫픽스

### 사진이 다른 유저에게 안 보임 (c7d89cd)
사용자 보고: "다른 분 사진 안 보여요". 원인: Supabase Image Transformation endpoint 응답 깨짐 가능성.
즉시 폴백: SUPABASE_IMAGE_TRANSFORM_ENABLED = false (3곳: lib/cats-repo, /map 마커, CareLogTab). 원본 URL 직접 사용 → public 버킷이라 RLS 통과 보장. 대역폭 ↑이지만 사진 보장.

### broadcast-dm photo_url 컬럼 누락 (7bbc3dd, 354e1d8)
초기 시도: og_200 안내 DM 204건 모두 실패. 진단용 debug(chunk=1 + firstError 응답)로 정확한 에러 노출:
`Could not find the 'photo_url' column of 'direct_messages' in the schema cache`
원인: route가 NULL 컬럼 포함. direct_messages 스키마에 없음.
수정: 필드 제거 + chunk 500 복원. 204건 전원 전송 성공.
