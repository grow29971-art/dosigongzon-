# 개발일지 2026-07-11 — 결제 마무리 · 토큰 중앙화 · 홈 리디자인 1~2차

## 요약

결제 시스템 잔여 보안(레이트 리밋)과 후원 상품 UX를 마무리하고, 정식 오픈 준비물
(법적 고지·상품 폼·사전알림)을 갖춘 뒤, 피그마 AI 시안 기반 홈 리디자인까지 진행.
커밋 15개+, 전부 프로덕션 배포. 라이브 결제는 사업자등록 후로 확정.

## 오늘 한 것

### 1. 결제/보안 마무리
- 레이트 리밋: 주문 생성(분당 5/일 30, enforceUserActionLimit) +
  결제 승인·취소 API(사용자당 분당 10, 인메모리 429)
- 토스 웹훅 등록은 사업자등록 → PG 심사 때 함께 진행하기로 결정 (코드는 완성 상태)

### 2. 후원(가상) 상품 UX
- 전 상품이 is_virtual이면 주문서에서 배송지·배송메모 생략 + 안내 배너
  (`supabase_shop_virtual_order_migration.sql` — 수령인/주소 NOT NULL 해제, 실행 완료)
- 주문 상세·관리자 주문에서 배송지 null → "후원 주문 (배송 없음)" 표시
- 후원 문구에서 퍼센트 숫자 제거 → "수익의 일부 후원" (운영 현실 반영)
- 상품 상세 후원 안내에 🛒구매→💰수익→💛일부 후원 미니 흐름 추가
  (거창한 구조도는 기부금품법 리스크 + 비율 변경 부담으로 기각)
- **후원 비율 20% → 10% 확정** (`supabase_shop_donation_10_migration.sql`, 실행 완료)
- /maker·/about 운영 문구 현행화 — "전부 자비 부담" → "자비 + 쇼핑몰 수익, 일부는 후원"

### 3. 정식 오픈 준비물
- `/shop/policy` — 판매자 정보(사업자번호 자리 비움)·배송·교환반품(7일 청약철회)·환불 고지
- 관리자 상품 폼 v2 컬럼 노출: 배지/후원 연계+비율/가상상품/무게/도매처 메모
- 쇼핑몰 오픈 사전알림 카드 (PushOptInCard에 title/description/dismissKey prop 추가)
- 하단 네비: 가이드 → **AI집사**(Bot 아이콘, /tips 유지), 쇼핑 → **쇼핑(공사중)**
- /tips 페이지를 AI집사 중심으로 재구성 (헤더·메타 변경, 챗봇 카드 최상단)

### 4. 관리자 도구
- cats 관리자 DELETE RLS 정책 추가 (`supabase_cats_admin_delete_migration.sql`, 실행 완료)
  — 기존엔 정책이 없어 관리자 삭제가 조용히 실패했음
- 지도 위치 변경도 관리자 허용 (기존 등록자 전용 2곳)
- `/admin/cats` 신규 — 검색·숨김 토글·체크박스 일괄 삭제 (더미 데이터 정리용)
- ⚠ 미해결: 사용자 계정에서 지도 수정/삭제 버튼이 안 보인다는 보고 —
  앱 로그인 계정이 admins 테이블에 있는지 확인 필요 (진단 SQL 전달해둠)

### 5. 디자인 토큰 중앙화 (비주얼 리뉴얼 기반)
- globals.css @theme: radius 6종·shadow 5종·primary 틴트 2종 토큰 추가
- 142개 파일 일괄 치환: hex 542건→var(), 그림자 229건→토큰, 라운드 76건→토큰
- 제외(의도): OG 이미지(Satori), 지도/캔버스 직렬화 파일, SVG 속성,
  `${색상}15` 알파 접미사 조합·DB 저장 색상 맵 10곳
- 이후 무드 변경 = globals.css 토큰만 수정하면 앱 전체 반영

### 6. 홈 리디자인 (피그마 AI 시안 선별 적용)
- **1차**: 헤더 레벨바 제거(중복), 날씨 위젯 압축, 내 활동 요약 기본 접힘,
  내 동네 소식 → 고양이/이야기 세그먼트 탭 1카드
- **2차**: 브랜드 헤더+날씨 → "오늘의 브리핑" 카드 통합
  (인사·🔥스트릭 칩·검색/알림·개인화 헤드라인 "OO가 밥을 기다려요"(실데이터)+날씨 행),
  플로팅 "🍚 돌봄 기록하기" 버튼(스크롤 따라다님, 탭→내 아이들 스크롤),
  스트릭 카드는 프리즈 보존 위해 하단 이동, "내 고양이"→"내 아이들"
- 시안 기각: 가짜 상태 배지(배고픔 등 — 데이터 없음), 스트릭 카드 완전 삭제
- 친구 초대 카드를 1000명 키링 배너 바로 위로 이동 (초대→보상 동선)

### 7. 협업 규칙 (메모리 저장)
- 냥줍(catchcat)은 별개 토이 프로젝트 — city 세션에서 언급 금지
- 되돌리기 항상 가능하게: 한 작업=한 커밋 유지, SQL 마이그레이션에 롤백 SQL 주석 동봉
