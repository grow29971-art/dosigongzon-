## 2. 고양이 프로필 강화

### 추가 컬럼
- `gender`: male / female / unknown
- `neutered`: true / false / null(모름)
- `health_status`: good / caution / danger
- `photo_urls`: text[] (추후 다중 사진용)

### UI 적용
- 등록 모달: 성별·중성화·건강 선택 버튼
- 카드 보기: 이름 아래 뱃지 (양호일 땐 건강 뱃지 숨김)
- 수정 모드: 기존 값 불러와서 편집

## 3. 홈 화면 리뉴얼

### 내 활동 요약 카드
- 갈색 그라디언트 배너 (레벨 이모지 + 칭호 + Lv 뱃지)
- 골드 경험치 바 + 점수/목표 표시
- 스탯 4칸: 고양이 / 돌봄 / 신고 / 좋아요

### 인기 커뮤니티 글
- 좋아요+댓글+조회수 가중 합산 상위 3개

### 날씨 기반 돌봄 팁
| 조건 | 팁 |
|------|-----|
| 체감 -10°C 이하 | 🥶 핫팩 + 물 자주 교체 |
| 체감 0°C 이하 | ❄️ 물 얼음 주의 + 🏠 숨숨집 |
| 체감 5°C 이하 | 🧣 쉼터 점검 |
| 체감 28°C+ | ☀️ 그늘 + 물 교체 |
| 체감 33°C+ | 🔥 폭염! 사료 변질 주의 |
| 비/눈 | 🌧️🌨️ 밥그릇 보호 |
| 습도 85%+ | 💦 건사료 눅눅 주의 |
| 바람 10m/s+ | 💨 무거운 그릇 사용 |
| 쾌적한 날 | 🐾 돌봄하기 좋은 날씨! |

## 4. AI 집사 개선

### 법적
- 면책 고지: "AI 답변은 참고용이며 전문 수의사 상담을 대체하지 않습니다"
- 데이터 고지: "대화 내용은 Google AI에 전송되며 서버에 저장되지 않습니다"
- 시스템 프롬프트 의료 가드레일: 약 이름/용량/처방 절대 금지

### 기능
- 빠른 질문 버튼 6개 (새끼 발견, 응급처치, TNR, 밥 주의사항, 병원 찾기, 쉼터)
- 레벨별 대화 제한 차등 (Lv1=10, Lv3=20, Lv5=30회/분)
- 병원 연계 안내 강화
