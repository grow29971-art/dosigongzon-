- CSS 변수 오버라이드 + Tailwind 클래스 + 인라인 style 대응
- 지도 페이지는 다크모드 제외 (`.no-dark` 클래스)
- BottomNav 별도 다크 처리
- 아이콘 박스: 라이트=연한 배경+컬러 아이콘 / 다크=다크그레이+흰색 아이콘

### 다크모드 시행착오
- CSS `filter: invert()` 방식 → `position: fixed` 깨짐 → 폐기
- `body > *` 필터 → BottomNav 사라짐 → 폐기
- 최종: CSS 변수 + 셀렉터 직접 오버라이드 방식 채택
- 인라인 style은 CSS selector보다 우선하므로 attribute selector 한계 있음
- `.dark-card-level`, `.dark-icon-box`, `.no-dark` 등 유틸 클래스로 해결

## 3. 인증/계정 기능
- **아이디·비밀번호 찾기** (`/find-account`)
  - 아이디 찾기: 닉네임 입력 → 마스킹 이메일 반환 (`gr*****1@gmail.com`)
  - 비밀번호 찾기: Supabase `resetPasswordForEmail()` 연동
  - Rate limiting: IP당 1분 5회 (브루트포스 방지)
- **회원탈퇴** (`/api/account/delete`)
  - Bearer 토큰 인증
  - "탈퇴합니다" 텍스트 입력 확인 (실수 방지)
  - `admin.deleteUser()` → CASCADE 삭제
- **닉네임 중복 방지** (`/api/check-nickname`)
  - 회원가입 시 + 마이페이지 닉네임 변경 시 모두 적용
  - 본인 현재 닉네임은 제외 (`currentUserId` 파라미터)
