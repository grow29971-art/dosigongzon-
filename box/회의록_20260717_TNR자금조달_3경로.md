# 회의록 — TNR 자금 조달 3경로 비교 (2026-07-17)

> 안건: 펀드투표에서 유저가 TNR을 사용처 1위로 뽑음(6표 중 3표). "돈을 어떻게 조달할 것인가"를 3경로로 비교.
> 참석: BM · B2G · 길고양이도메인 · 리서치(법률/수치) · 악마의변호인 (5명)
> 실측 전제: WAU 6, 좀비 79%, orders 0건, 펀드 잔액 ≈0원, 실지불의향 신호 부재.

## 3경로
- **A. 타깃 후원 캠페인** — 특정 고양이 TNR 목표금액·진행률 못박아 직접 후원.
- **B. 지자체 TNR 예산 연계(B2G)** — 지자체 집행 예산의 데이터/기록 레이어로 연계.
- **C. 커머스 판매마진 일부 → 펀드** — 쇼핑몰 라이브로 마진 일부 후원.

## 🟢 만장일치 결론: A 우선 (단, 지금은 "최소 실험"만), C 탈락, B 중장기 포석

## 핵심 논거
- **C 탈락 (악마·BM·리서치)**: 팔린 게 0인데 "마진 일부"는 0의 일부. 조달원이 아니라 매출이 나야 작동하는 부차 파이프. 지금 시간 쓰면 순수 환상 노동.
- **B는 12~18개월 (B2G)**: 예산 주기상 7월 접촉해도 2027년 집행. 지자체가 보는 건 유저 수가 아니라 **단일 자치구 개체 밀도 데이터**. 지금 할 건 코드가 아니라 **데이터 export + PDF 1장으로 담당 주무관 미팅 1건 시도**. 미팅 안 잡히면 아직 이른 것. 신규 기능 개발 시작하면 그게 도피.
- **A가 유일하게 지금 지불의향을 싸게 실측 (BM·악마)**: 목표는 조달액이 아니라 **"실제 송금 1건" 발생 여부**. 뜨면 B 제안서 근거로, 안 뜨면 "대의명분도 안 팔린다"는 냉정한 데이터 확보.

## A경로 실행 조건 (도메인 + 법률)
- **첫 대상**: 코어 다수가 이미 아는 **단골 미중성 암컷**(임신 방지 명분 = 시민·지자체 설득력 최상). 낯선 신규 개체는 실패.
- **신뢰 4요소(캣맘 국룰)**: 병원 견적서 → 목표액·마감 → 집행 후 72h 내 영수증 → 실시간 잔액 공개. 미달성 시 전액 이월/반환 명문화. 사연은 care 로그로 검증 가능한 것만(과장 금지).
- **자산 재활용**: 고양이 프로필=캠페인 대상, care 로그=진정성 증빙, fund_disbursements=집행 공개. **단 현 정산 테이블은 펀드 전체 단위 → 개체별 campaign_id·목표액 필드 없음(신설 필요).**
- **법적 안전판 (리서치)**:
  - 목표 1천만원 미만이면 기부금품법 등록 면제(TNR 소액이라 세이프). 그래도 목적 외 사용 금지·투명집행 의무.
  - **자체 앱 직접 수령은 리스크 전부 창업자가 짊어짐** → 초기엔 **플랫폼(카카오같이가치·네이버 해피빈) 경유** 또는 **송금은 외부 계좌, 앱은 기록·투명공개만** 하이브리드가 안전. (결제 하드락 미해제 상태에서 실화폐 직접 수령 = 통신판매업 위반 리스크)
  - TNR 실비: 지자체 단가 ~19.5만/마리, 개인 자비 수컷 20~40만·암컷 35~70만(지역 편차, 지자체 보조 후 포획·이동은 자부담).

## ⏱️ 타이밍 (악마 — 시기상조 반론 채택)
지금 본격 펀딩 구축은 이르다. 옳은 시점 = **인터뷰 18명 답(7/20) 집계 + WAU가 진짜 두 자릿수 회복 후.** "6표 = 일화지 표본 아님. 말(투표)이지 행동(orders 0) 아님. 행동이 이긴다."

## ✅ 지금 당장 할 것 (빌드 최소, 리텐션·유입과 무충돌)
1. **인터뷰 질문지에 1문항 추가** (코드 0): "TNR에 실제로 O천원 낼 의향 있으세요?" → 말이 아닌 지불의향 직접 타진.
2. **(선택) 고양이 상세에 "이 아이 TNR 후원 관심?" 관심 카운트 버튼 1개**(결제 아님, 카운트만) — 지불의향을 공짜로 관측. 단 이것도 빌드이므로 인터뷰 문항이 1순위.
3. **펀드투표 + 투명정산 UI는 유지** — 커머스가 아니라 신뢰·관계 기능. 잔액 0이어도 코어에게 "내가 정한 곳에 투명하게"라는 감각 제공.
4. **(B 포석) 코어 활동 자치구 1곳 개체 데이터 PDF 1장** 준비 → 나중 구청 미팅용. 지금은 만들어만 두기.

2. 관심 버튼/인터뷰에서 실지불의향 신호(말 아닌 행동)가 잡힌다.
