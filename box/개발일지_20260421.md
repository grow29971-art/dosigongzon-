## 4. 활동 지역 설정 지도 빈 회색 이슈 수정 (피드백)

- 증상: 유저들이 `/mypage/activity-regions`에서 검색 후 지도가 빈 회색 화면
- 원인 분석:
  1. 컨테이너 크기 늦게 잡혀서 타일 미로드
  2. `script.onerror` 처리 누락 → SDK 로드 실패 시 무한 대기
  3. SDK load timeout 없음
- 수정:
  - `map.relayout()` 150ms·500ms·1500ms 3단계 자동 호출
  - `ResizeObserver` 로 컨테이너 크기 변화 감지 시 relayout
  - `resize`/`orientationchange` 이벤트 연결
  - 컨테이너 minHeight 280px 보장
  - SDK script `onerror` + 15초 timeout → 친절한 에러 UI (새로고침 버튼)
