## 1. 추가 검색채널 등록 (예상 30분~1시간)

### Bing 웹마스터 — ⭐️ 가장 우선
- 링크: https://www.bing.com/webmasters
- 단계:
  1. Microsoft 계정 또는 Google 계정으로 로그인
  2. "Import from Google Search Console" 선택 → 자동으로 사이트·sitemap 가져옴
  3. URL 검사 도구로 메인 페이지 한번 fetch
- 효과: ChatGPT 검색·Copilot이 Bing 인덱스 사용. AI 검색 시대에 점점 중요해짐

### Daum 검색등록
- 링크: https://register.search.daum.net
- 단계:
  1. "신규 등록" → 사이트 정보 입력
  2. 사이트 URL: `https://dosigongzon.com`
  3. 카테고리: 사회/문화 > 환경 또는 동물
  4. 심사 1~2주 소요
- 효과: 다음 검색에서 노출. 점유율 작지만 노년층/지방 유입 있음

### Zum 사이트 등록
- 링크: https://help.zum.com/submit
- 단계: 사이트 정보 폼 작성 후 제출
- 효과: 미미하지만 한 번 등록해두면 영구 유효

### 추가로 고려해볼 것
- **줌 학생** (학생 대상): N/A
- **Yandex Webmaster**: 한국 사용자엔 의미 없음, 패스

## 2. 백링크 만들기 (효과 큼 · 시간 걸림)

### 나무위키 외부 링크 — ⭐️ ROI 최고
- 대상 항목: "길고양이", "TNR", "동물보호법", "캣맘"
- 단계:
  1. 위 항목들 "관련 사이트" / "외부 링크" 섹션 확인
  2. 도시공존 관련 페이지 링크 추가 (예: 보호지침, areas)
  3. 편집 사유에 "관련 시민 플랫폼 추가" 명시
- 주의: 광고성으로 보이면 즉시 revert됨. 항목 본문에 자연스럽게 녹이거나 외부 링크 섹션에 한 줄로.
- 효과: 영구 백링크 + 직접 트래픽 (나무위키 트래픽 자체가 큼)

### 네이버 카페 공유 — ⭐️ 한국 검색에 직접 영향

  - 지역별 캣맘 카페 (강남, 송파, 마포 등 검색)
