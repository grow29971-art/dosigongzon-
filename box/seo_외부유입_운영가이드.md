## 1. Google Search Console 점검 (주 1회, 10분)

URL: https://search.google.com/search-console

### 첫 점검 시 확인
- [ ] `dosigongzon.com` 속성 등록되어 있는지 (이미 layout.tsx에 verification 코드 있음)
- [ ] **Sitemaps**: `https://dosigongzon.com/sitemap.xml` 제출 상태 확인. "성공" 안 뜨면 재제출
- [ ] **색인 생성 보고서**: 색인된 페이지 수 vs 발견됨-색인 안 됨. 색인 안 된 이유 확인
- [ ] **검색 실적**: 노출수·클릭수·CTR. 어떤 키워드로 노출되는지 → 콘텐츠 보강 단서

### 신규 페이지 색인 요청
신규로 만든 페이지는 Google이 발견할 때까지 1~4주 걸림. 즉시 색인 요청 가능:
- `https://dosigongzon.com/faq` ← 방금 만든 페이지
- `https://dosigongzon.com/feed.xml` ← RSS (직접 색인은 안 되지만 Google 발견 신호)

좌측 메뉴 **URL 검사** → 위 URL 입력 → **색인 생성 요청**

### 주간 점검 루틴
- 노출 늘었는지 (전주 대비)
- CTR 떨어진 키워드 → title/description 다듬기
- 404·5xx 오류 페이지 → 수정 또는 redirect

## 2. 네이버 웹마스터 도구 (월 1회, 5분)

URL: https://searchadvisor.naver.com

### 확인
- [ ] 사이트 등록·인증 (layout.tsx에 `naver-site-verification` 메타 이미 있음)

## 4. 공식 SNS 운영 — 인스타그램 1개부터

### 왜 인스타그램?
- 한국 길고양이·반려동물 콘텐츠 가장 활발한 채널
- 사진 1장 + 짧은 글로 진입 부담 적음
- 해시태그로 신규 유저 유입 (검색 기반)
- 사이트 OG 이미지가 매력적이라 공유 효과 큼

### 초기 운영 가이드

**계정 설정:**
- 이름: `도시공존 | 길고양이 시민 지도` (검색용)
- 사용자명: `dosigongzon` (도메인과 동일)
- 소개: "서울 25개 구 길고양이 시민 참여 지도 / 광고 없는 비영리 / 매일 새 친구 등록 중"
- 링크: `dosigongzon.com`
- 카테고리: 비영리 단체

**콘텐츠 카테고리 (주 3~5회):**
1. **이번 주 신규 등록 고양이** — 동네·이름·짧은 이야기 (운영자 큐레이션, 본인 동의 받은 경우)
2. **꿀팁 한 장 카드뉴스** — 사이트 /tips 콘텐츠를 카드 1~3장으로 요약
3. **법·뉴스 요약** — /news 콘텐츠 핵심만
4. **TNR·구조 케이스 스토리** — 익명 처리 후 한 화면씩
5. **동물숏츠 하이라이트** — 인스타 릴스로 재업로드 (출처 표기)

**해시태그 (게시물 1건당 15~25개):**
```
#길고양이 #길냥이 #캣맘 #캣대디 #TNR #길고양이돌봄 #서울길고양이
#동물보호 #구조동물 #임시보호 #길고양이입양 #동물보호법
#도시공존 #나의캣맘이야기 #우리동네고양이
```

**스레드(Threads) 동시 운영:**
- 인스타와 통합 계정. 글로 짧은 코멘트·뉴스 인사이트 공유
- 검색엔진 색인되므로 SEO 백링크 효과도

### sameAs JSON-LD 연결
공식 SNS 계정 만들면 코드로 연결 가능 — 알려주시면 Organization schema에 `sameAs` 배열로 추가해서 Google이 공식 채널로 인식하도록 처리.

## 6. 측정 — KPI

| 지표 | 목표 | 측정 방법 |
|---|---|---|
| Google 노출수 (월간) | 매월 +30% | Search Console |

| 인스타 팔로워 | 첫 3개월 500명 | 인스타 인사이트 |
| 사이트 직접 방문 | 매월 +20% | Vercel Analytics |
| 핵심 콘텐츠 페이지뷰 (/faq, /protection/*) | 매월 +30% | Vercel Analytics |

### 측정 도구
- **Google Search Console** — 검색 기반 (무료)
- **Vercel Analytics** — 사이트 내 트래픽 (이미 layout.tsx에 통합)
- **Speed Insights** — 페이지 속도 (이미 통합)
- **Ahrefs Webmaster Tools** — 백링크 추적 (무료 티어 있음)

## 참고: 코드 측 인프라 (이미 적용됨)

본 가이드는 다음 코드 인프라 위에서 동작:
- `sitemap.ts` — 정적·동적 URL 모두 색인
- `robots.ts` — Yeti(네이버) 별도 규칙
- `layout.tsx` — Organization·WebSite JSON-LD, Google·Naver verification, RSS link
- 페이지별 metadata + BreadcrumbList JSON-LD (13페이지)
- `/cats /community /tips /news /shorts /areas /protection` 모두 동적 OG + 카톡 공유 버튼
- `/faq` (FAQPage schema) + `/feed.xml` (RSS 2.0)
