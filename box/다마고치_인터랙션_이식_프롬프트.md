# [프롬프트] 다마고치 "쓰다듬기" 인터랙션 이식 (냥줍 게임용)

> 도시공존에서 검증·배포된 사양을 범용화한 것. 냥줍 캐릭터가 이미지·스프라이트·SVG 무엇이든
> `data-react` 하나로 그대로 적용된다. 아래를 통째로 냥줍 작업 에이전트에 붙여넣어 구현한다.

## 목표
캐릭터(고양이)를 탭할 때마다 **① 골골(purr) 소리 ② 골골 맥동 진동 ③ 주변 하트 뿅뿅 ④ 12종 중 랜덤 움직임**이
즉시(서버 왕복과 무관하게) 나오게 한다.

## 통합 지점
- 캐릭터 탭 핸들러(`onClick`/`onTouchStart`)와 "쓰다듬기" 버튼 핸들러 **양쪽**에서 `petTap()` 하나를 호출한다.
- `petTap()`은 즉시 실행 — 보상 API 응답을 기다리지 않는다(손맛). 보상 로직은 별개로 둔다.
- 움직임 애니메이션은 **캐릭터 자신의 엘리먼트**(img/svg/sprite)에 걸어, 부모의 idle(숨쉬기)과 겹치지 않고 합성되게 한다.

## 수용 기준
1. 탭할 때마다 직전과 **다른** 움직임이 나온다(연속 반복 없음).
2. 움직임이 끝난 뒤 idle(숨쉬기) 애니메이션이 그대로 이어진다.
3. 소리는 폰 스피커에서 또렷하게 들린다(저역 롤오프 대응).
4. 진동은 모바일(안드로이드 크롬 등 `navigator.vibrate` 지원)에서 골골처럼 맥동한다.
5. 하트 6개가 약간의 시차를 두고 캐릭터 주변에 떠오른다.
6. `prefers-reduced-motion`에서 반응 애니메이션은 꺼진다.
7. 공용 음소거 설정이 있으면 소리는 그것을 따른다(진동/움직임은 유지).

## 코드 1 — 골골 사운드 (자립형, WebAudio 합성 · 오디오 파일 불필요)

```ts
// purr.ts — 오디오 파일 없이 저역 럼블 + 27Hz 트레몰로로 "그르르르" 합성.
// 폰 스피커 로우롤오프 감안해 몸통을 180Hz대에 둔다. 음소거 훅은 프로젝트에 맞게 연결.
let _ctx: AudioContext | null = null;
let _reverb: ConvolverNode | null = null;

function getCtx(): AudioContext | null {
  if (typeof window === "undefined") return null;
  if (!_ctx) {
    const C = window.AudioContext ?? (window as unknown as { webkitAudioContext?: typeof AudioContext }).webkitAudioContext;
    if (!C) return null;
    _ctx = new C();
  }
  // iOS: 사용자 제스처 안에서 resume 되도록 탭 콜백에서 primePurr()도 호출할 것
  if (_ctx.state === "suspended" && document.visibilityState === "visible") _ctx.resume();
  return _ctx;
}
function getReverb(ctx: AudioContext): ConvolverNode {
  if (_reverb) return _reverb;
  const len = Math.floor(ctx.sampleRate * 0.55);
  const buf = ctx.createBuffer(2, len, ctx.sampleRate);
  for (let ch = 0; ch < 2; ch++) {
    const d = buf.getChannelData(ch);
    for (let i = 0; i < len; i++) d[i] = (Math.random() * 2 - 1) * Math.pow(1 - i / len, 2.2);
  }
  _reverb = ctx.createConvolver(); _reverb.buffer = buf;
  return _reverb;
}

/** iOS 대응: 캐릭터 탭 콜백 맨 앞에서 한 번 호출해 오디오 컨텍스트를 제스처로 활성화. */
export function primePurr() { getCtx(); }

/** 쓰다듬기 골골송. isMuted()가 true면 무음(프로젝트 음소거와 연결). */
export function playPurr({ duration = 1.1, volume = 0.34 }: { duration?: number; volume?: number } = {}) {
  // if (isMuted()) return;               // ← 냥줍 음소거 훅이 있으면 주석 해제
  const ctx = getCtx();
  if (!ctx) return;
  const t0 = ctx.currentTime;

  // 몸통 — 저음(배음 풍부) + 옥타브 아래 두께감
  const osc = ctx.createOscillator();  osc.type = "sawtooth"; osc.frequency.setValueAtTime(184, t0);
  const osc2 = ctx.createOscillator(); osc2.type = "triangle"; osc2.frequency.setValueAtTime(92, t0);

  // 숨결 텍스처 — 브라운 노이즈
  const len = Math.max(1, Math.floor(ctx.sampleRate * duration));
  const buf = ctx.createBuffer(1, len, ctx.sampleRate);
  const data = buf.getChannelData(0);
  let last = 0;
  for (let i = 0; i < len; i++) { const w = Math.random() * 2 - 1; last = (last + 0.02 * w) / 1.02; data[i] = last * 3.2; }
  const noise = ctx.createBufferSource(); noise.buffer = buf;

  const lp = ctx.createBiquadFilter(); lp.type = "lowpass"; lp.frequency.value = 1100; lp.Q.value = 0.7;

  // 골골 맥동 — 27Hz LFO로 진폭 변조(트레몰로)
  const trem = ctx.createGain(); trem.gain.setValueAtTime(0.6, t0);
  const lfo = ctx.createOscillator(); lfo.type = "sine"; lfo.frequency.setValueAtTime(27, t0);
  const lfoDepth = ctx.createGain(); lfoDepth.gain.setValueAtTime(0.4, t0);
  lfo.connect(lfoDepth).connect(trem.gain);

  // 엔벨로프
  const env = ctx.createGain();
  env.gain.setValueAtTime(0.0001, t0);
  env.gain.linearRampToValueAtTime(volume, t0 + 0.14);
  env.gain.setValueAtTime(volume, t0 + Math.max(0.2, duration - 0.25));
  env.gain.exponentialRampToValueAtTime(0.0001, t0 + duration);

  osc.connect(lp); osc2.connect(lp); noise.connect(lp);
  lp.connect(trem).connect(env);
  env.connect(ctx.destination);
  const send = ctx.createGain(); send.gain.value = 0.18;         // 살짝 리버브(공간감)
  env.connect(send).connect(getReverb(ctx)).connect(ctx.destination);

  const stopAt = t0 + duration + 0.05;
  osc.start(t0); osc2.start(t0); noise.start(t0); lfo.start(t0);
  osc.stop(stopAt); osc2.stop(stopAt); noise.stop(stopAt); lfo.stop(stopAt);
}
```

## 코드 3 — React 배선 (탭 즉시: 소리+진동+움직임+하트)

```tsx
import { playPurr, primePurr, PURR_VIBE } from "./purr";

const N_REACT = 12; // 움직임 종류 수

// 컴포넌트 안
const [reactVariant, setReactVariant] = useState(-1);           // -1 = 아직 없음
const [fx, setFx] = useState<{ id: number; e: string; x: number; y: number }[]>([]);
const fxId = useRef(0);

// 매번 직전과 다른 변형을 고름(같은 값이면 CSS 애니메이션이 재생 안 되므로 필수)
const nextReact = () => setReactVariant((v) =>
  v < 0 ? Math.floor(Math.random() * N_REACT)
        : (v + 1 + Math.floor(Math.random() * (N_REACT - 1))) % N_REACT);

const spawnFx = (e: string, x: number, y: number) => {
  const id = ++fxId.current;
  setFx((p) => [...p, { id, e, x, y }]);
  setTimeout(() => setFx((p) => p.filter((f) => f.id !== id)), 1200);
};

// 하트 뿅뿅 — 6개를 시차 두고 캐릭터 주변(퍼센트 좌표)에 팡팡
const heartBurst = () => {
  const hearts = ["💕", "💗", "💖", "❤️", "💛"];
  for (let i = 0; i < 6; i++) {
    const x = 50 + (Math.random() * 46 - 23);   // 27~73%
    const y = 36 + (Math.random() * 24 - 8);    // 캐릭터 상체 주변
    setTimeout(() => spawnFx(hearts[i % hearts.length], x, y), i * 55);
  }
};

// 탭 즉시 실행 — 캐릭터 onClick과 "쓰다듬기" 버튼 양쪽에서 호출
const petTap = () => {
  primePurr();
  playPurr();
  try { navigator.vibrate?.(PURR_VIBE); } catch {}
  nextReact();
  heartBurst();
};
```

```tsx
))}
```

## 핵심 포인트 (구현 시 주의)
- **매번 다른 움직임의 비결**: 같은 `data-react` 값을 다시 넣으면 CSS 애니메이션이 재시작되지 않는다.
  `nextReact()`가 항상 직전과 다른 값을 보장하므로 속성이 바뀌며 재생된다.
- **idle과의 합성**: 반응은 **캐릭터 엘리먼트**에, 숨쉬기 idle은 **부모(.cat)**에 걸면 서로 다른 요소라
  반응 후 idle이 자연히 이어진다. 같은 요소에 두 애니메이션을 넣으면 반응이 끝난 뒤 idle이 안 돌아온다.
- **transform-origin: 50% 100%**(발끝) 필수 — 점프/회전이 자연스럽다.
- **하트 좌표**는 씬 컨테이너(`position:relative; overflow:hidden`) 기준 퍼센트다.
- **음소거**: 냥줍에 공용 sfx 음소거가 있으면 `playPurr` 맨 앞에서 그걸 확인하도록 연결.
