# 학습 로드맵 — 팔리는 제품을 위한 8주 파운데이션

**작성일:** 2026. 04. 10
**출처:** 외부 강의 커리큘럼 (Curriculum.pdf)
**관련 프로젝트:** 도시공존 (본 리포지토리)
**문서 성격:** 개인 학습 계획. 본 프로젝트의 제품 요구(PRD) 및 기술 요구(TRD)와는 별개로 관리한다. 학습 과정에서 얻은 지식 중 도시공존에 반영할 항목이 생기면 그때 TRD에 공식 편입한다.

## 과정 목표

> "장난감이 아니라, 돈을 받을 수 있는 '프로덕트'의 뼈대를 만든다."

단순히 화면만 띄우는 것을 넘어, 유지보수가 가능하고 확장성 있는 서비스를 만들기 위해 현업 수준의 기술 스택(Modern Stack)을 학습한다. 2개월 동안 기초 공사를 끝내고, 이후 스스로 살을 붙여나갈 수 있는 자생력을 기른다.

---

## 과정 개요

- **기간:** 총 8주 (2개월) + 이후 지속적 디벨롭 권장
- **일정:** 주 1회, 2시간 수업 (숙제 강도 높음)
- **기술 스택 (Pro Stack):**
  - **Language:** TypeScript (필수, 타입 안정성)
  - **Framework:** Next.js (App Router)
  - **UI Library:** shadcn/ui + Tailwind CSS (디자인 시스템)
  - **Backend:** Supabase (DB & Auth)
  - **API Layer:** tRPC (Type-safe API, 프론트엔드-백엔드 자동 연결)

## 도시공존 프로젝트와의 연관성

### 이미 도입된 항목
- **TypeScript** — 전 코드베이스 적용 중
- **Next.js (App Router)** — `app/` 디렉토리 기반으로 구현 중. 단, 현재는 **Next.js 16** (커리큘럼 기준 15 → 실제 프로젝트는 한 버전 위)
- **Tailwind CSS** — 적용 중 (2026-04-10 세이지/클레이 팔레트로 전면 리뉴얼)
- **Supabase (DB & Auth)** — 이메일/OAuth 인증 연동 완료

### 학습 후 도입을 검토할 항목
- **shadcn/ui** — 현재는 `lucide-react` 아이콘 + 직접 작성한 Tailwind 컴포넌트 중심. 2주차 학습 이후 일부 페이지부터 점진적으로 마이그레이션을 검토. TRD 1.1 스택표에 공식 편입됨 (2026-04-10).
- **React Hook Form + Zod** — 현재 회원가입/로그인 폼은 `useState` 기반 수동 검증. 4주차 학습 이후 재작성 검토.
- **tRPC** — **도입 여부 신중히 판단 필요.** 현재 TRD는 `Drizzle ORM + Next.js Route Handler` 구조로 이미 타입 안전성을 확보하고 있어, tRPC를 추가하면 레이어가 중복되고 기존 코드를 상당량 리팩토링해야 한다. 학습 목적으로 별도 토이 프로젝트에서 경험을 쌓은 뒤, 도시공존 v2 착수 시점에 재평가한다.
- **Prisma ORM** — 현재 Drizzle ORM을 채택했기 때문에 도입 불필요. 학습용으로만 경험.
