- [ ] 대표자 신분증

## 신고 절차 (정부24, 수수료 등록면허세 별도)
1. [ ] 정부24(gov.kr) 접속 → "통신판매업 신고" 검색
2. [ ] 신고서 작성: 상호·사업자번호·소재지·**인터넷 도메인(dosigongzon.com)**·호스트서버 소재지(Vercel — 해외 소재로 기재)
3. [ ] 구매안전서비스 이용확인증 첨부
4. [ ] 관할 구청 접수 → 처리(통상 3일 내) → **신고번호 수령**
5. [ ] 등록면허세 납부 (지자체별 연 12,000~40,500원)

## 함께 밀린 것 (참고 — project_legal_compliance 메모리)
- LBS(위치기반서비스) 신고 — GPS 기능 켜기 전제조건. 통신판매업 신고하러 갈 때 같이 알아보면 효율적
- 면세 여부: 간이과세자면 세금계산서 의무 없음. 연 매출 규모상 간이 유지가 유리할 가능성 높음 (세무 상담 권장)
