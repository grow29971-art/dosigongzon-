영향: 신규 방문자 체류시간↑, SEO 핵심 가이드 도달률↑, Google 페널티 위험 해소.

## 2. 버그 진단 → CRITICAL 4건 일괄 픽스

병렬 에이전트 2개로 (1) 보안·데이터 무결성, (2) 런타임·UX 진단. CRITICAL 4건 발견 → 모두 픽스.

| # | 항목 | 위치 |
|---|---|---|
| 1 | **JSON-LD XSS** — `JSON.stringify` 결과를 그대로 `dangerouslySetInnerHTML`. 댓글/고양이 이름에 `</script>` 넣으면 스크립트 탈출 | 20개 파일 / 40개 호출처 |
| 2 | **storage `cat-photos` insert 정책 누락** — 다른 유저 폴더로 업로드/덮어쓰기 가능 | `box/supabase_cats_schema.sql` |
| 3 | **auth-context 구독 cleanup 누수** — `import().then()` 안에서 cleanup 반환이 React에 안 닿음 | `lib/auth-context.tsx` |
| 4 | **한글 IME 조합 중 Enter** — `isComposing` 가드 없음. "안녕하세ㅛ" 같은 미완성 댓글 제출됨 | community/[id], map, mypage, mypage/activity-regions |

기술 메모:
- JSON-LD XSS 픽스는 perl 일괄치환 시도 → bash↔perl 백슬래시 escape 단계가 어긋나 `<` → `003c`로 변형되는 사고. Node 스크립트(`box/fix_jsonld_escape.mjs`)로 1회용 fix 스크립트 작성해 일괄 복구. 최종 파일에 `"\\u003c"` (2 백슬래시) 정상 진입 확인 (od -c 검증).
- IME 가드는 `!e.nativeEvent.isComposing` 한 줄 추가.

## 3. 키링 응모 수 표시 (Event1000Banner.tsx)

홈 1000명 배너에 가입자 수 외에 **현재 응모자 수**도 표시 — 자기 동네 사람들이 많이 응모했다는 사회적 증명.

## 4. 캣맘 활동 랭킹 시스템 (신규)

목표: "동네 1위" 같은 영역 자존심 동기로 retention 강화.

작업:
- **SQL RPC 함수 2개** (`box/supabase_ranking_function_migration.sql`)
  - `get_top_caretakers(limit_n)` — 점수 + 순위 매겨진 상위 N명
  - `get_caretaker_rank(target_user_id)` — 본인 순위 + 전체 인원
  - SECURITY DEFINER + anon/authenticated 둘 다 grant
- **점수 산식**: `cat × 10 + comment + alert × 2 + likes_received × 2 + care_log × 2`
- **`lib/ranking-repo.ts`** — 서버 RPC 래퍼
- **`/ranking` 페이지** — Top 3 시상대(메달 + 그라디언트) + 4~50위 리스트 + 본인이 50위 밖이면 별도 표시 + 점수 산식 접기 안내 + 비로그인 가입 유도. 10분 ISR.
- **마이페이지 진입점** — "🏆 캣맘 활동 랭킹" 메뉴 카드 추가

v1은 "전체 랭킹"만. 향후 "내 동네", "주간" 추가 예정.
