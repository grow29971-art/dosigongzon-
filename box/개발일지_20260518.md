### 5. Circle RLS 무한 재귀 핫픽스 (a14353b)
**버그**: 지도 빈 화면 — "infinite recursion detected in policy for relation caretaker_circles".

**원인**: `caretaker_circles` ↔ `circle_members` SELECT 정책이 서로 EXISTS로 참조 → cats 정책에서 JOIN으로 호출 시 양쪽 정책이 무한 재귀.

**해결 (`box/supabase_circle_recursion_fix.sql`)**: security definer 함수 3개로 RLS 우회.
- `is_circle_member_of(owner_id)` — cats 정책의 circle 멤버십 체크용
- `is_circle_owner_of(circle_id)` — circle_members 정책의 owner 체크용
- `has_any_membership_of(circle_id)` — 초대받은 유저의 owner 정보 조회용

기존 5개 정책 drop → 함수 3개 생성 → 정책 4개 재생성. Dashboard 실행 완료.
