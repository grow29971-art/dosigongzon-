|------|------|
| `lib/supabase/client.ts` | 브라우저용 Supabase 클라이언트 |

### 수정 파일
| 파일 | 변경 내용 |
|------|------|
| `app/layout.tsx` | AuthProvider로 앱 전체 감싸기 |
| `app/login/page.tsx` | Supabase 이메일/비밀번호 로그인 + Google/Kakao OAuth 연동 |
| `app/signup/page.tsx` | Supabase 이메일 회원가입 + 이메일 인증 안내 화면 추가 |
| `app/(main)/mypage/page.tsx` | 로그인 상태에 따른 유저 정보 표시 + 로그아웃 기능 |
| `package.json` | `@supabase/supabase-js`, `@supabase/ssr` 추가 |
| `.gitignore` | `.vercel` 디렉토리 제외 추가 |

### 구현 내용
- **이메일/비밀번호 회원가입**: 닉네임 + 이메일 + 비밀번호로 실제 Supabase 회원가입
- **이메일/비밀번호 로그인**: Supabase `signInWithPassword` 연동
- **소셜 로그인 UI**: Google, Kakao OAuth 플로우 구현 (Provider 활성화 필요)
- **AuthProvider 컨텍스트**: `useAuth()` 훅으로 앱 전체에서 유저 상태 접근
- **마이페이지**: 로그인 시 닉네임/이메일/아바타 표시, 비로그인 시 게스트 + 로그인 유도
- **로그아웃**: 마이페이지에서 로그아웃 버튼 클릭 시 세션 제거
- **이메일 인증 안내**: 회원가입 후 "인증 메일을 보냈습니다" 화면 표시
- **에러 핸들링**: 잘못된 비밀번호, 이미 가입된 이메일 등 한국어 에러 메시지
- **Suspense 처리**: 로그인 페이지 `useSearchParams()` Suspense boundary 적용
- **환경변수 미설정 대응**: Supabase 키 없이도 빌드 가능하도록 graceful 처리
