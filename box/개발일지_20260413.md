## 1. 지도 전면 개편

### 마커 색상 분류
| 태그 | 색상 |
|---|---|
| 학대 경보 | 🔴 빨강 + 펄스 |
| TNR 필요 | 🟠 주황 |
| TNR 완료/이어팁 | 🟢 초록 |
| 기본 | 🟤 갈색 |

### 동 단위 클러스터
- 개별 고양이 핀 → **동별 그룹 마커** ("🐾 만수동 5")
- 대표 사진 최대 3장 겹침 표시
- 클러스터 탭 → 해당 동 고양이 목록 패널 → 개별 선택 → 상세

### 고양이 등록 개선
- 좌표 입력 시 **카카오 역지오코딩** → 구/동 자동 감지
- 동 드롭다운 자동 생성 + 직접 입력 병행

### 고양이 수정/삭제
- 본인 등록 고양이 + admin: ✏️ 수정 / 🔴 삭제 버튼
- 인라인 편집 (이름/설명/동네/태그)
- `updateCat()`, `deleteCat()` 함수 추가

### 학대 경보 구/동 단위
- **지도 viewport 기반** 필터링 → 현재 보이는 구의 경보만 표시
- "남동구 학대 경보 2건" + 동별 뱃지 (`도림동 1건`, `구월동 1건`)
- 다른 구로 이동 → 해당 구 경보로 자동 갱신
