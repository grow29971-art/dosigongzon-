## §2. 코드 픽스 3건 (weather·push 배포됨 / dm는 SQL)
- 🟡 **weather API env 노출 제거**(배포): 키 미설정 응답에서 `OPENWEATHERMAP_API_KEY`·`.env.local` 힌트 삭제, 원인은 `console.error`로만. (0368a14)
- 🟡 **push/send userId UUID 검증**(배포): `hasLegitRelation`의 `.or()`에 body userId가 문자열 보간되던 latent injection 차단. UUID 정규식 불일치 시 400. (0faf1f9)
- 🟡 **dm UPDATE 하드닝 SQL**(실행 대기): `dm_update_read`에 `with check` 없어 수신자가 body 위조 가능. `box/supabase_dm_update_hardening_migration.sql` — with check + 컬럼 UPDATE grant(is_read/read_at만). 의존성 없어 언제든 실행 OK. (54d9746)
