# 인터뷰 판정 기준 — 집계(7/20) 전 사전 등록 (2026-07-18)

> **왜 이 문서인가**: 7/18 전체회의에서 악마의 변호인 지적 — "판정 기준을 미리 못 박지 않으면,
> 가장 확률 높은 시나리오(응답 5명 안팎의 우호적·애매한 답변)가 '희망적 신호'로 해석되고
> 2주 뒤 똑같은 회의를 반복한다." 이 문서는 **집계 결과를 보기 전에** 기준을 고정하는 사전 등록이다.
> 집계 후 이 문서를 수정하지 말 것. 기준 변경이 필요하면 사유와 함께 새 섹션으로 추가.

## 0. 표본의 한계 (해석 전 필수 라벨)
- 발송 18명 = 이미 활성·우호적 유저 표본 → **이중 편향(생존+우호)**. 응답은 상한선(best case)으로만 읽는다.
- 지불의향 답변은 stated preference — 실제 결제 전환의 **상한선**이지 예측치가 아니다.
- 무응답도 데이터다: 무응답자 수·특성(가입 시기, 마지막 활동일)을 반드시 기록.

## 1. 응답률 게이트 (판정 가능 여부)
| 응답 수 | 판정 |
|---|---|
| **8명 이상** | 정성 판정 진행 (아래 §2~3) |
| **4~7명** | 부분 판정 — "강한 신호"만 채택, 애매한 답변은 전부 무신호 처리 |
| **3명 이하** | **판정 불가 선언.** 희망적 해석 금지. 리마인드(7/21~22 1회) 후 7/24 재집계. 그래도 3명 이하면 "인터뷰 채널 실패"로 기록하고 §4의 대체 신호로 전환 |

## 2. TNR 지불의향 (5천원 문항) 판정
기존 기준(인터뷰_리마인드_문구_20260717.md §5) 유지 + 수치 고정:
- **양성**: "낸다"를 구체적으로(왜/언제/어떤 아이면) 말한 사람 **3명 이상**, 그중 코어(주 1회 이상 활동) **1명 이상 포함**
- **음성**: "좋은데 잘 모르겠다" / "마음은 있는데…" 류만 → 지불의향 없음 (말≠행동)
- **판정 결과 → 행동**:
  - 양성 → TNR 타깃 후원형 상품 설계 착수 (단, WAU 두 자릿수 회복과 AND 조건 — 하나만으로 본격화 금지)
  - 음성 → **사료 커머스·후원형 모두 보류.** Stage 4 결제 개발 재개하지 않음. 다음 BM 실험 설계로
- ⚠️ 만장일치 양성이어도 **결제 오픈의 근거로 승격 금지** — BM 계산상 커머스 수학(월 1.6만원 상한)은 인터뷰 결과와 무관하게 성립하지 않는다. 결제 오픈은 별도 의사결정.

## 3. 이탈 원인 (자유 답변) 판정
- 같은 계열 원인이 **3명 이상**에서 반복되면 "주 원인 후보"로 채택 → 2주 내 1개만 수정
- 1~2명 언급은 기록만 (n=소표본에서 단발 의견에 개발 리소스 배정 금지)
- "좋아요/잘 쓰고 있어요"만 있는 답변은 **무신호로 집계** (예의 필터)

## 4. 인터뷰와 무관하게 이미 결정된 것 (재론 금지)
- 통신판매업 신고: 인터뷰 결과와 무관하게 진행 (무손실 선행 — 사장님 보류 중, 7/18)
- 핸드오프+퍼널 계측: 배포 완료(658f28a) — 효과 판정은 인터뷰가 아니라 funnel_events 숫자로
- Stage 4 결제 개발: 인터뷰 양성이어도 자동 재개 아님 (신고 완료 + 별도 결정 필요)

## 5. 인터뷰 실패 시 대체 신호 (§1 판정 불가 케이스)
- funnel_events 퍼널 수치 (intro→pick→signup_home→first_feed) — 신규 유입이 있는 한 자동 축적
- 주간 신규 가입 수 실측 (회의 사각지대 — 획득 채널이 살아있는지) — service_role 쿼리
